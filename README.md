# Sales & Demand Forecasting Project

## Project Description
This project predicts future sales using historical retail data from the Superstore dataset. 
It uses **Python (Linear Regression & Prophet)** for forecasting and **Power BI** for visualization. 
The forecasts help businesses make better inventory, staffing, and sales decisions.

## Technologies Used
- Python
- Pandas, NumPy, scikit-learn
- Prophet (Meta/Facebook)
- Matplotlib
- Power BI

## Project Features
- Data cleaning and preprocessing
- Feature engineering (date, month, year)
- Linear Regression sales forecast
- Prophet time-series forecast for next 6 months
- Power BI dashboard for interactive visualization
